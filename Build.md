## Instructions

    ./configure
    make
    build/dino

## Dependencies
**Build-time**

* CMake
* C compiler
* gettext
* ninja(-build) (recommend)
* valac (≥ 0.30)

**Run-time**

* GLib (≥ 2.38)
* GTK (≥ 3.22)
* GPGME (For the OpenPGP plugin)
* libgee-0.8 (≥ 0.10)
* libgcrypt (For the OMEMO plugin)
* libnotify
* SQLite3
